隐藏用户建立

1. 以 $ 结尾的隐藏用户

​ 用户名要以$结尾

net user hello$ 123456 /add

​ 此时运行“ net user ” 命令是看不到新建的” hello$ “ 用户的。

​ 但是可以在用户管理界面查看到刚刚建立的用户，也可以切换用户查看到该账号

2. 隐藏用户管理界面影子账号的显示

2.1 导出注册表

​ 打开注册表

win + R => regedit

​ 这里需要先给“ HKEY_LOCAL_MACHINE\SAM ” 添加一下权限，然后重启注册表

依次展开[ HKEY_LOCAL_MACHINE\SAM\SAM ]->[右键]->[权限] 在[组或用户名]栏中选中[ Administrators ]勾选[完全控制]和[读取]然后[确定]

​ 重启注册表后在” HKEY_LOCAL_MACHINE\SAM\SAM\Domains\Account\Users\Names “ 下找到” hello$ “查看他的默认类型值

​ 根据注册表中 “ hello$ “账号下的 type 导出同级目录下相对应的文件夹(总共需要导出两个文件: hello$ 和 000003ED 文件)

​ 右键文件夹点击 “ export (导出)”

​ 导出的文件以” .reg “结尾

​ 导出文件后就可以删除账号了

net user [: username] /del

2.2 导入reg文件到注册表

进入导出注册表表项的路径

cd desktop

重新将表项导入到注册表中

regedit /s hello.reg

regedit /s hello.reg

此时在注册表中又能看到 hello$,运行“ net user hello$ ”也能查询该账户的基本信息

在用户管理界面已经查看不到刚刚建立的用户

3. 深度隐藏

​ 这时候 hello$ 可以进行登录，但是登录后同样会在本地计算机用户目录下产生用户文件夹信息

​ 这时候可以将注册表项” HKEY_LOCAL_MACHINE\ SAM\ SAM\ Domains\ Account\ Users\ 000003ED (创建的隐藏用户的文件夹)”的 F 键值替换为系统中其他的能正常登录的用户的键值即可

​ 通常是替换管理员administrator的键值。默认情况下，管理员帐户Administrator对应的注册表键值为” HKEY_LOCAL_MACHINE\ SAM\ SAM\ Domains\ Account\ Users\ 000001F4 “

​ 这时候登录admin$用户共用administrator账户文件系统，将不记录用户自身文件信息。相当于一个影子账户。

注意：也可以在用户创建的时候 net user [: username] /home:[: path] 指定家目录达到隐藏的效果
