windows 提权 建立隐藏用户(net user 命令)

0. net user 概述

​ “net user”是Windows操作系统中的命令行实用程序，用于管理用户账户。通过执行”net user”命令，您可以查看、创建、修改和删除用户账户。该命令提供了一些选项，如添加用户、更改密码、启用/禁用账户等。您可以在命令提示符下输入”net user”来查看有关该命令的详细用法和选项。

1. net user 使用

1.1 net user 查看当前用户

net user

1.2 net user 添加用户

1.2.1 添加普通用户

net user [: username] [: password] /add

1.2.2 添加域用户

net user [: username] [: password] /add /domain:[: domain_name]

1.3 net user 删除用户

net user [: username] /del

1.4 net user 查看用户详细信息

net user [: username]

1.5 net user 激活用户

net user [: username] /active:yes 激活

1.6 net user 设置用户访问时间

net user [: username] /times:weekday1-weekday2:time1-time2

​ 设置 [: username] 用户 weekday1 到 weekday2 早上 time1 到 time2 可以访问

日期必须为:

  • Monday (周一)
  • Tuesday (周二),
  • Wednesday (周三)
  • Thursday (周四)
  • Friday (周五)
  • Saturday (周六)
  • Sunday (周日)

​ 可以使用缩写, 像周六必须写成 Sa, 周日必须写成 Su

1.7 net user 设置用户到期时间

net user [: username] /expires:year/month/day hour:minute:second

1.8 net user 设置用户根目录

net user [: username] /home:[: path]

1.9. net user 是否允许用户自己更改密码

net user [: username] /passwordchg:yes|no

1.10 net user 指定用户是否必须设置密码

net user [: username] /passwordreq::yes|no

1.11 net user 指定用户配置文件路径

net user [:username] /profilepath:[: path]

1.12 net user 使管理员可以更改此账户的描述信息

net user [:username] /usercomment:”text”

1.13 net user 为账户设定描述信息

net user [:username] /comment:”text”

1.14 net user 指定主机名能够登录此账户

net user [:username] /workstations:computername[,…] | *
